Pré-requisito — Antes de Começar

Instale a extensão Allow CORS no Chrome

As imagens de satélite (NDVI e NDMI) do Sentinel-2 são carregadas diretamente da API da ESA. Por questões de segurança, o navegador bloqueia esse tipo de requisição entre origens diferentes (política CORS). Para liberar o carregamento das imagens, instale a extensão gratuita abaixo no Google Chrome:

↗ Instalar Allow CORS — Chrome Web Store
  1. Clique no link acima e depois em Usar no Chrome
  2. Após instalada, clique no ícone da extensão (canto superior direito do Chrome)
  3. Ative o toggle para habilitar a extensão
  4. Recarregue o OrbitAgro Copilot — as imagens de satélite carregarão normalmente

Obs: A extensão só precisa estar ativa durante o uso do site. Você pode desativá-la a qualquer momento pelo mesmo ícone.

  1. 01
    Etapa 01 — Configuração

    Identifique sua Lavoura

    Informe o nome do seu talhão e selecione a cultura predominante (Soja, Milho, Cana, Café ou Algodão). O Copilot carrega automaticamente o perfil agronômico da cultura: tipo de solo ideal, práticas de manejo sustentável, ranking de produção nacional e as pragas mais comuns da região.

    ↳ O sistema usa a geolocalização do seu dispositivo para identificar a região
  2. 02
    Etapa 02 — NDVI

    Monitore a Saúde Foliar via Satélite

    O sistema busca a imagem mais recente do satélite Sentinel-2 (ESA) para a sua coordenada. O mapa NDVI codificado em cores revela áreas com vegetação densa e saudável (verde intenso), zonas de deficiência (amarelo/marrom) e solo exposto (cinza). Dados de temperatura e qualidade do ar local complementam o panorama.

    ↳ Imagens atualizadas a cada 5 dias com resolução de 10 metros
  3. 03
    Etapa 03 — NDMI

    Analise o Balanço Hídrico

    O mapa NDMI (Índice de Umidade) exibe o teor de água no solo e na vegetação. Zonas azuis indicam saturação hídrica; marrons indicam seca severa. Combinado com a previsão de chuva para os próximos 7 dias e a porcentagem de precipitação esperada, o Copilot orienta a estratégia de irrigação.

    ↳ Previsão meteorológica atualizada em tempo real pela API Open-Meteo
  4. 04
    Etapa 04 — Catálogo Biológico

    Identifique Ameaças no Campo

    Explore o catálogo interativo de pragas específicas da sua cultura. Clique nos pontos luminosos sobrepostos à lavoura para visualizar fotos do inseto e dos danos causados na planta, com descrições técnicas dos sintomas. Essa etapa prepara o produtor para a observação de campo e para o preenchimento do Laudo.

    ↳ Catálogo com as principais pragas para cada uma das 5 culturas suportadas
  5. 05
    Etapa 05 — Diagnóstico

    Gere o Laudo e Receba Recomendações

    Preencha o Laudo Analítico com suas observações de campo: saúde foliar aparente, presença de zonas críticas no mapa hídrico, dias sem chuva e nível de infestação de pragas. O algoritmo Copilot pondera todos os dados e gera um Diagnóstico Final com nível de risco e uma lista priorizada de ações agronômicas.

    ↳ Diagnóstico: NOMINAL / ATENÇÃO / CRÍTICO — com parecer técnico por escrito

Fontes de Dados

Todas as informações utilizadas pelo Copilot são públicas, abertas e confiáveis.

🛰️

Copernicus / ESA — Sentinel-2

Programa espacial europeu de observação da Terra. As imagens multiespectrais são disponibilizadas gratuitamente para uso científico e comercial. Fornecem as bandas NIR e SWIR para cálculo de NDVI e NDMI.

🌦️

Open-Meteo API

API meteorológica open-source com dados de 80+ modelos climáticos globais. Fornece temperatura atual, qualidade do ar, probabilidade de precipitação e previsão para 7 dias com resolução de 1km.

🗺️

Embrapa — Base Agronômica

Os perfis de solo, práticas de manejo e dados de pragas foram compilados a partir de publicações técnicas da Empresa Brasileira de Pesquisa Agropecuária (Embrapa), a maior instituição de pesquisa agrícola do Brasil.

📍

Geolocation Web API

API nativa do navegador (W3C) que captura as coordenadas GPS do dispositivo do produtor com permissão explícita. Nenhuma coordenada é armazenada em servidores — o processamento é feito localmente.
