O Projeto

OrbitAgro Copilot

Inteligência climática e produtiva para pequenos e médios produtores. Um copiloto que traduz dados de satélite, clima e sinais da lavoura em ações práticas — em linguagem simples.

HTML5 CSS3 JavaScript ES6+ WebGL / GLSL Sentinel Hub OpenWeather Gemini AI Geolocation API FIAP GS 2026/1
Nossa Missão
"O campo brasileiro não precisa de mais dados — precisa de dados que falem a língua de quem planta."

O OrbitAgro Copilot foi criado para fechar a lacuna entre a agricultura de precisão e o produtor rural que não tem acesso a consultores ou equipamentos caros. Combinamos imagens de satélite, previsão climática e inteligência artificial para entregar uma orientação que qualquer produtor consiga entender e agir — em menos de dois minutos.

Por que isso importa

O agronegócio brasileiro é gigante. As ferramentas de precisão, ainda inacessíveis para a maioria.

4.9M

estabelecimentos agropecuários no Brasil que podem se beneficiar do monitoramento por satélite

30%

redução potencial em perdas de produção quando pragas são detectadas antes dos sintomas visuais

5 dias

frequência de atualização das imagens Sentinel-2, permitindo monitoramento contínuo da lavoura

⚠️

O que trava a produtividade

Pequenos e médios produtores enfrentam perdas significativas quando o momento certo de agir não fica claro. Os principais fatores:

  • Seca e chuva imprevisível, tornando o planejamento difícil
  • Atraso na identificação de pragas e estresse hídrico
  • Falta de leitura simples do que o clima realmente indica
  • Dificuldade em interpretar dados de satélite sem especialistas
  • Exclusão das ferramentas de precisão por custo elevado
🛰️

O que o OrbitAgro entrega

Combinando cultura selecionada, dados climáticos, imagens de satélite e sintomas observados, o sistema entrega orientações práticas e priorizadas:

  • NDVI e NDMI via satélite: identifica variações e áreas em risco
  • Clima em tempo real: interpreta previsões para os próximos dias
  • Catálogo de pragas: identificação visual por cultura
  • Copilot de IA: recomendações claras geradas pelo Gemini
  • Interface acessível: sem necessidade de treinamento técnico

Stack Tecnológico

100% front-end, sem frameworks. HTML semântico, CSS modular e JavaScript puro integrados a APIs de classe mundial.

🌐

Frontend

HTML5 semântico, CSS3 modular e JavaScript ES6+. Nenhum framework externo — layout responsivo construído do zero com Grid e Flexbox nativos.

✨

WebGL + Canvas

Motor de partículas e animações em WebGL com shaders GLSL customizados. Chuva, folhas e estrelas renderizadas sem Three.js ou bibliotecas externas.

🧠

Gemini AI

Google Gemini 1.5 Flash para o Copilot agronômico. Processa dados de satélite, clima e pragas e gera diagnósticos em linguagem acessível ao produtor.

🌦️

OpenWeather API

Dados climáticos em tempo real — temperatura, umidade, precipitação e previsão de 5 dias para a coordenada exata da lavoura do produtor.

🛰️

Sentinel Hub

Imagens multiespectrais dos satélites Sentinel-2 da ESA. Resolução de 10 m/pixel para mapas NDVI (vegetação) e NDMI (umidade foliar) em tempo real.

📍

Geolocation API

API nativa do navegador para capturar latitude e longitude do dispositivo. Geolocalização automática da lavoura sem configuração manual pelo produtor.

Desenvolvido na FIAP como resposta ao desafio semestral de maior escala da instituição.

O Desafio

A FIAP Global Solution é um hackathon semestral onde turmas de diferentes cursos desenvolvem soluções tecnológicas para problemas reais de alta complexidade. No primeiro semestre de 2026, o tema central é segurança alimentar e agricultura sustentável — desafios críticos para o Brasil e o mundo.

O OrbitAgro Copilot é a resposta da turma 1TDSPF ao desafio: como democratizar o acesso à agricultura de precisão para quem mais precisa?

Critérios de Entrega

Regras do Projeto

A FIAP exige que toda a solução seja construída com tecnologias fundamentais — sem Bootstrap, jQuery, React ou Vue. Cada linha de CSS e JavaScript foi escrita do zero, comprovando domínio real das bases da web moderna.

  • Frontend puro: sem frameworks ou bibliotecas de UI
  • CSS modular com variáveis nativas e custom properties
  • JavaScript ES6+ sem dependências externas de terceiros
  • APIs reais integradas com fetch e tratamento de erros
  • Deploy estático funcional em ambiente de produção
