Global Solution 2026/1 — FIAP

A Solução OrbitAgro

Inteligência orbital a serviço da segurança alimentar. Dados de satélite, clima e IA unidos para proteger a lavoura brasileira.

⚠️

O Problema

O Brasil enfrenta uma contradição: é uma das maiores potências agrícolas do mundo, mas milhões de pequenos e médios produtores operam sem acesso a ferramentas de precisão. O resultado:

  • Perdas de até 30% da produção por pragas identificadas tardiamente
  • Desperdício de água e insumos por irrigação sem dados reais
  • Decisões baseadas em intuição em vez de dados espaciais
  • Exclusão digital do campo da revolução da agricultura 4.0
🛰️

Nossa Proposta

O OrbitAgro Copilot democratiza o acesso à agricultura de precisão. Em vez de equipamentos caros e técnicos especializados, o produtor acessa pelo navegador e obtém:

  • Mapa de saúde foliar NDVI via satélite Sentinel-2
  • Mapa hídrico NDMI com zonas de deficit e saturação
  • Previsão meteorológica local em tempo real
  • Catálogo interativo de pragas por cultura
  • Diagnóstico agronômico com recomendações práticas

Tecnologias Utilizadas

Stack 100% front-end, sem frameworks — HTML semântico, CSS modular e JavaScript puro.

🌍

Sentinel-2 / ESA

Satélites da Agência Espacial Europeia com resolução de 10m/pixel. Fonte das imagens NDVI e NDMI para monitoramento da cobertura vegetal e umidade do solo.

🌦️

Open-Meteo API

API meteorológica de código aberto com dados climáticos em tempo real: temperatura, qualidade do ar (AQI), precipitação e previsão para os próximos 7 dias.

📍

Geolocation API

API nativa do navegador para capturar a latitude e longitude do dispositivo do produtor, permitindo a geolocalização automática da lavoura sem configurações manuais.

✨

WebGL / Canvas

Motor de partículas renderizado em WebGL via Canvas 2D para criar o ambiente visual imersivo do sistema, sem bibliotecas externas como Three.js.

🧠

Motor de IA Local

Algoritmo de diagnóstico agronômico em JavaScript puro, que pondera dados de satélite, clima, pragas e solo para gerar recomendações priorizadas.

📐

CSS Grid + Flexbox

Layout totalmente responsivo com CSS puro — sem Bootstrap ou qualquer framework. Breakpoints para mobile (≤480px), tablet (≤768px) e desktop (≥992px).
